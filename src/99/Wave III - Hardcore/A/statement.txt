Sorting
    Sorting is one of the most important topics in computer science. There  are
    several types of sorting based on comparison, counting and so on.

Problem Statement:
    In this task you are asked to sort an array of tuples  with  some  specific
    key.
    Arr = [ (x,y), ...]
    Key = (x-y)**2


Input and Rules:
    $n:int
    (1 < n < 100)
    [ n times (x, y) ]
        (-10000 < x, y < 10000)
    
Sample:
    Input:
    3
    -1 1 3 2 4 9
    Output:
    3 2 -1 1 4 9
